Bias Detective Investigation
Scenario
You're a QA engineer at TalentMatch AI, a Ghanaian HR tech company. The company has developed "HireScore," an AI tool that ranks job applicants for client companies. After 6 months of use, several clients have complained that the tool seems to favor certain types of candidates unfairly. You've been assigned to investigate potential bias in the system.

Your Investigation Tasks
Task 1: Identify Bias Types

For each bias type below, determine if it exists in HireScore and provide evidence:

1. Historical Bias

Definition: Bias from past societal inequalities reflected in training data

Present in HireScore? (Yes/No)

Evidence from artifacts:

Which groups are disadvantaged?

2. Sampling Bias

Definition: Training data doesn't represent the full population

Present in HireScore? (Yes/No)

Evidence from artifacts:

Which groups are underrepresented?

3. Measurement Bias

Definition: Features or labels are measured differently across groups

Present in HireScore? (Yes/No)

Evidence from artifacts:

What's being measured unfairly?

4. Proxy Bias

Definition: Features correlate with protected attributes (gender, ethnicity, region)

Present in HireScore? (Yes/No)

Which features act as proxies?

How do they disadvantage certain groups?

Task 3: Propose Mitigation Strategies

For each category, recommend specific, actionable solutions:

Immediate Actions (Can be done this week):

Feature removal: Which features should be removed and why?

Threshold adjustments: Should scoring thresholds be adjusted?

Output monitoring: What metrics should be tracked weekly?

Short-Term Actions (Next 1-3 months):

Data collection: What new training data should be gathered?

Model retraining: What should change about the training approach?

Human oversight: Where should human review be added?

Long-Term Actions (Next 6-12 months):

Fairness metrics: Which fairness definition should TalentMatch adopt? (Demographic parity, Equalized odds, Equal opportunity)

Process changes: What should change about how clients use HireScore?

Transparency: What should be disclosed to applicants and clients?
